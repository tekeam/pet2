# NinjaPet Independent Platform

اسکریپت مستقل NinjaPet بدون وردپرس و بدون بخش فروشگاه داخلی. این سرویس شامل پروفایل کاربر، پروفایل پت، پیشنهاد غذا، برندها، مدل‌های غذا، مراکز پت، اکسپلور، پنل مدیریت و ورود ایمیلی/پیامکی است.

## اجرا

```bash
npm install
npm run build
npm start
```

آدرس برنامه:

```text
http://localhost:3001
```

پنل مدیریت:

```text
/#/admin
```

ورود پیش‌فرض مدیر:

```text
admin
admin
```

## امکانات

- ثبت‌نام با ایمیل، شماره موبایل، یا هر دو بر اساس تنظیمات پنل مدیریت
- اتصال SMS.ir برای ارسال OTP با API Key و Template ID
- پنل مدیریت کامل با تغییر رمز مدیر
- برندها و مدل‌های غذای خشک سگ و گربه به صورت seed اولیه
- لینک خرید برای هر مدل غذا
- پیشنهاد غذا بر اساس مشکل پت
- پروفایل کاربر و چند پت برای هر کاربر
- تقویم یادآور فارسی به صورت متنی
- ثبت مراکز پت شامل پت‌شاپ، دامپزشکی، آرایشگر و پانسیون
- تایید مراکز و پست‌های اکسپلور از پنل مدیریت
- بکاپ JSON از کل داده‌ها
- دیتای استان و شهرهای ایران

## نکته SMS.ir

برای SMS.ir در پنل مدیریت این موارد را وارد کنید:

- API Key
- Template ID / Pattern ID
- فعال‌سازی گزینه SMS.ir

در حالت توسعه اگر SMS فعال نباشد، کد OTP در پاسخ API برمی‌گردد تا تست راحت باشد.

## فایل دیتابیس

داده‌ها در فایل زیر نگهداری می‌شوند:

```text
data/db.json
```

برای ریست کردن کامل، این فایل را حذف کنید و سرور را دوباره اجرا کنید.
